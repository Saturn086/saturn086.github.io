Overview

Back in 2015 I took the video game development course as an introduction to Java through game design and programming. This game was the final course project and was developed by a team of four, including myself.

The game is set on the 1920's as you, an eager archaeologist, delve into a recently discovered tomb just to find yourself trapped in a maze full of monsters. The main objective is to escape by either fighting or evading monsters, it is a game of action-packed exploration.

The game has over 7,000 lines of code since it was developed from scratch using only an animation library. The theme, gameplay and feel to it was inspired by Edmund McMillen and Florian Himsl's The Binding of Isaac.

Features

  • Build entirely on pure Java code, no specialized libraries were used.
  • Built using JFrames.
  • Features 3 types of monsters and one final boss, each with a unique attack pattern and behavior.
  • Levels are generated randomly using a maze creation algorithm.
  • The game comes with an original soundtrack and artwork.
